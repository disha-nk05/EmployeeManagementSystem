# Employee Management System

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap
- Chart.js

## Features
- User Login System
- Employee CRUD (Add, Edit, Delete)
- Search Employees
- Dashboard with Charts
- Role-based UI (Admin/HR)
